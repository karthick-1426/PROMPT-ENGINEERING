#  Types of Generative AI Models

## 1. Generative Adversarial Networks (GANs)

GANs consist of:

### Generator

Creates synthetic data.

### Discriminator

Distinguishes real data from generated data.

### Applications

* Image Generation
* Video Creation
* Face Synthesis

---

## 2. Variational Autoencoders (VAEs)

VAEs learn compressed representations of data and generate new variations.

### Applications

* Style Transfer
* Image Generation
* Data Compression

---

## 3. Transformer Models

Transformers revolutionized AI by processing entire sequences simultaneously.

### Applications

* ChatGPT
* Gemini

* Translation Systems

---

## 4. Restricted Boltzmann Machines (RBMs)

* Feature Learning

#  Transformer Architecture

The Transformer architecture was introduced in 2017 through the paper:

> **"Attention Is All You Need"**

It is the foundation of modern Large Language Models.

---

## Advantages of Transformers

### Scalability

Processes large datasets efficiently.

### Parallel Processing

Handles entire sequences simultaneously.

### Context Awareness

Captures relationships between distant words.

---

#  Attention Mechanism

The Attention Mechanism helps the model focus on the most relevant parts of the input.

### Benefits

* Better Context Understanding
* Improved Accuracy
* Enhanced Language Generation

---

#  Tokenization and Encoding

Before processing text, Transformers convert words into tokens.

### Tokenization

```text
"The robot learns quickly"
↓
["The", "robot", "learns", "quickly"]
```

Each token is assigned a numerical value.

---

## Encoding

Tokens are transformed into vectors that capture semantic meaning and relationships.

---

#  Positional Encoding

Transformers require positional information to understand word order.

### Example

```text
The robot analyzes data.
```

Word order influences meaning.

Positional Encoding helps preserve this order.

---

#  Vectorization

Words are represented as vectors in high-dimensional space.

### Example

```text
Robot ≈ Machine
```

Similar words have similar vector representations.
